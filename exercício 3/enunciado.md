3. Validação de Regras: Triângulos
Objetivo: Testar lógica booleana composta (operadores and / or).

Função: tipo_triangulo(a, b, c) que retorna "Equilátero", "Isósceles" ou "Escaleno".

Atividade: Criar testes para os três tipos.

Desafio: Adicionar um teste para um caso onde os valores não formam um triângulo (ex: 1, 1, 10), garantindo que a função retorne uma string de erro ou "Invalido".

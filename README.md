# Testes Automatizados com Pytest

Repositório com exercícios desenvolvidos para praticar testes automatizados em Python utilizando `pytest`.

## Exercícios

### 1. Teste de Média Simples
Teste da função de cálculo de média utilizando `assert`.

### 2. Sistema de Notas
Testes para verificar aprovação, exame e reprovação com estruturas condicionais.

### 3. Validação de Triângulos
Testes para verificar diferentes tipos de triângulos e casos inválidos.

### 4. Teste de Desconto em Compras
Testes de valores limite para aplicação de desconto em produtos.

### 5. Área do Círculo
Testes para validar cálculo de área e tratamento de valores negativos.

## Assuntos praticados

- Testes automatizados
- `pytest`
- Uso de `assert`
- Validação de funções
- Testes de condições
- Boundary testing
- Estruturas condicionais
- Lógica de programação
